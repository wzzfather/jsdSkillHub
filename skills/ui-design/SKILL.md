# UI 设计 Skill

> **使用者**：PM AI（设计方案） + Cursor Agent（代码实现）
> **职责**：页面美工设计、组件规范、视觉一致性

---

## 设计原则

### 核心风格
- **企业级**：专业、简洁、高效，不花哨
- **一致性**：全局统一的配色、字体、间距、圆角
- **可定制**：支持多套主题切换（亮色/暗色），支持企业品牌色替换
- **响应式**：桌面端优先，兼顾平板和移动端

### 配色体系（默认主题）

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#404040` | 深灰，专业感（Element Plus primary） |

| 危险操作按钮 | `#e74c3c` | 红色，仅删除、驳回 |
| 辅助按钮 | 白色背景 + `#d1d5db` 灰边框 | 取消、返回等 |
| 页面背景 | `#f5f7fa` | 浅灰，不刺眼 |
| 卡片背景 | `#ffffff` | 纯白 |
| 文字主色 | `#1f2937` | 深灰黑 |
| 文字辅助色 | `#6b7280` | 中灰 |
| 成功 | `#22c55e` | 绿色标签 |
| 警告 | `#f59e0b` | 橙色标签 |
| 错误 | `#e74c3c` | 红色标签 |

## 组件规范

### 按钮使用规则

| 场景 | 按钮类型 | 样式 |
|------|---------|------|
| 新建/提交/通过/发布 | 主操作 | 绿色实心 `#22c55e` |
| 删除/驳回 | 危险操作 | 红色实心 `#e74c3c` |
| 取消/返回 | 次要操作 | 白色 + 灰边框 |
| 编辑/查看 | 普通操作 | 文字按钮或灰色边框 |
| 下载/导出 | 辅助操作 | 灰色边框 |

### 表格规范
- 斑马纹开启（`stripe`）
- 表头背景 `#f5f7fa`，字重 600
- 操作列固定在右侧
- 超长文本截断 + tooltip
- 空数据状态显示友好提示

### 表单规范
- 标签宽度：100px
- 必填项标红星
- 输入框高度：36px
- 错误提示在输入框下方，红色小字
- 表单底部固定操作按钮栏（保存 + 取消）

### 状态标签颜色

| 状态 | 背景色 | 文字色 |
|------|--------|--------|
| 待审核 | `#fef3c7` | `#92400e` |
| 已通过 | `#dcfce7` | `#166534` |
| 已驳回 | `#fee2e2` | `#991b1b` |
| 已上架 | `#dbeafe` | `#1e40af` |
| 已下架 | `#f3f4f6` | `#4b5563` |
| 扫描中 | `#e0e7ff` | `#3730a3` |

## 页面模板

### 列表页模板
```
面包屑导航
页面标题 + 右侧操作按钮（新建等）
搜索栏 + 筛选条件 + 批量操作
数据表格（或卡片视图切换）
分页器
```

### 详情页模板
```
面包屑导航
页面标题 + 操作按钮（编辑/删除/返回）
Tab 页签（基本信息 / 版本历史 / 扫描报告 / 审批记录）
每个 Tab 下的内容区域
```

### 审批页模板
```
待审批数量统计（顶部卡片）
待审批列表（表格形式）
每行：应用名称 + 提交者 + 提交时间 + 扫描状态 + 操作（查看详情 / 通过 / 驳回）
审批详情弹窗：左侧应用信息 + 右侧扫描报告 + 底部审批意见输入 + 通过/驳回按钮
```

## 与开发者（Cursor）的交付物

PM AI 设计完成后，向 Cursor 提供以下交付物：
1. **页面结构描述**：每个页面包含哪些区域、组件
2. **交互说明**：点击、hover、弹窗等行为
3. **配色/样式参数**：具体的色值、间距、圆角
4. **响应式断点**：桌面 ≥1024px / 平板 ≥768px / 手机 <768px
5. **状态流转图**：审批状态对应的 UI 变化

Cursor 必须严格按此规范实现，不得自行发挥设计。
