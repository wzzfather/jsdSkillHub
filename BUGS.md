## Bug 模板

复制以下格式新增 Bug：

```
### Bug: [简短描述]
- **模块**：xxx
- **发现日期**：YYYY-MM-DD
- **复现步骤**：
  1. xxx
  2. xxx
- **期望结果**：xxx
- **实际结果**：xxx
- **优先级**：P0 / P1 / P2 / P3
- **状态**：待修复 / 修复中 / 已修复 / 无法复现
- **修复记录**：（修复后填写）
```
